FIVE COLORS.
FIVE SHAPES.

Data-viz is the densest surface in the system. It gets the harshest rules. A closed 5-color palette reading --color-chart-1…5. Five chart shapes total: readout, sparkline, bar, stacked bar, heatmap. Mono type at 10–11px for axis and legend. Zero decoration, zero gradient fill. If your chart needs a 6th color or shape, your chart is two charts.

Rules of data viz

  • Five colors, closed. --color-chart-1…5. Use 1 before 2 before 3 — order is semantic, not decorative.
  • Five shapes, closed. Readout, sparkline, bar, stacked bar, heatmap. Pie, donut, radar, 3D, streamgraph — never.
  • No gradient fill, no drop shadow. Area charts use flat tone at <12% alpha, or they're line charts pretending.
  • Mono 10–11px for everything non-visual. Axis, legend, labels. Body type never enters the chart.
  • Grid fades to near-nothing. oklch(0.25 0 0 / 0.5), baseline only. No major/minor split.
  • A chart with >5 series is two charts. Split it or use small multiples. Never add a sixth palette color.
