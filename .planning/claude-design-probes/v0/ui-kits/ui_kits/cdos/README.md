# cdOS — internal operator console

Internal tooling surface for Culture Division. Dense two-column chrome: coded sidebar, toolbar with hue swatches, and a working pane that swaps between:

- `ComposerView` — signal canvas + token manifest + inspector with live `--signal-intensity`
- `QueueView` — ingest queue, data-dense mono table
- `TokensView` — color token ledger

Dark-only. No window chrome (this is embedded inside a desktop host). Sidebar groups use §-numbered headings per SF coded-nomenclature convention.
