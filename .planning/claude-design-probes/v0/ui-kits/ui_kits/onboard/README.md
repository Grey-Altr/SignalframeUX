# SF Onboard — generative onboarding

Three-step ingest flow that produces a themed SignalframeUX instance from (1) one primary hue + (2) five short answers. The agent synthesizes complementary tokens, seed, intensity, tracking, and preset name deterministically from the inputs.

- `§00 · PRIMARY` — swatch field, 36-step hue ring plus named presets (350 magenta is the default, not the prescribed choice).
- `§01 · INTERROGATION` — register, lineage, density, motion, surface. Two columns, coded keys.
- `§02 · SYNTHESIS` — generated manifest, barcode seed, live instance preview, and `manifest.css` export ready to hand to the user's agent.

The "HAND TO AGENT" / "APPLY TRADITIONALLY" buttons are the two branches the brief called for: agentic generation vs. classical design use of the system.
