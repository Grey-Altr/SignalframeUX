API_REFERENCE

Programmable design. Every token, component, and field is controllable via a single API. Built for deterministic visual systems.

createSignalframeUX(config: SignalframeConfig): SignalframeInstance

Interfaces // SignalframeConfig

// Configuration object for library initialization
interface SignalframeConfig {
target: HTMLElement | string;
theme?: 'dark' | 'light' | 'auto';
seed?: number;
density?: 0 | 0.5 | 1;
motion?: boolean;
}

Parameters

Name Type Default Description
target string null DOM selector or element to attach the renderer
theme enum 'dark' Initial visual theme mode
seed number Math.random() Determines the parametric pattern generation
