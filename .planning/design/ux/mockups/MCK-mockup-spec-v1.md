## Animation Patterns

| Pattern | Implementation | Guard |
|---------|---------------|-------|
| Entrance choreography | GSAP timeline, reading order (eyebrow → headline → body → CTA) | DOMContentLoaded |
| Scroll reveals | CSS `animation-timeline: view()` | `@supports (animation-timeline: scroll())` — fully visible fallback |
| Scroll reveals fallback | GSAP ScrollTrigger with `once: true` | Browsers without CSS scroll-driven animations |
| Spring physics | `cubic-bezier(0.34, 1.56, 0.64, 1)` on transforms | All interactive elements |
| Variable font animation | `font-weight: 400 → 700` transition on nav links | Inter variable font (wght axis 100-900) |
| Reduced motion | `prefers-reduced-motion: reduce` kills all durations | Global override |

## Accessibility Features

- Skip-to-main-content link on all screens
- Semantic HTML landmarks (header, nav, main, footer)
- `aria-label` on all interactive elements
- `:focus-visible` distinct from `:hover` on all controls
- `aria-busy`, `aria-disabled`, `aria-invalid` for state communication
- `aria-pressed` on theme toggle
- `aria-expanded` on disclosure toggles
- `role="status"` with `aria-live="polite"` for availability indicator
- Dark mode respects `prefers-color-scheme`
- `prefers-reduced-motion` disables all animations
