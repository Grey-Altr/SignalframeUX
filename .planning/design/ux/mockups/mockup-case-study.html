SignalframeUX Design System Hero

Design portfolios rarely demonstrate engineering depth. I needed a token-first design system that could power an entire portfolio while serving as proof-of-work for systematic design engineering.

  • WCAG 2.2 AA minimum
  • Zero JavaScript for core layout
  • OKLCH perceptual uniformity
  • Progressive enhancement

Phase 03

Engineering Execution

The token system compiles from a single JSON source to CSS custom properties, with automatic dark mode inversion, contrast-checked semantic aliases, and typed TypeScript exports.

tokens.json → tokens-to-css → tokens.css + tokens.ts

Performance was non-negotiable. The entire token system adds less than 4KB gzipped, with zero layout shift and sub-100ms paint times on 3G throttled connections.

Lighthouse: Performance 99 | Accessibility 100 | Best Practices 100
