BUTTON

SIGNAL LAYER · v2.1.0 · STABLE

THE PRIMARY INTERACTIVE ELEMENT. SUPPORTS SIGNAL (DETERMINISTIC) AND FIELD (GENERATIVE) VARIANTS. PROGRESSIVE ENHANCEMENT — FIELD EFFECTS LAYER ON TOP WITHOUT BREAKING ACCESSIBILITY.

IMPORT

import { Button } from '@sfux/components' // OR DIRECT IMPORT import { Button } from '@sfux/components/Button'

PROPS

PROPTYPEDEFAULTDESC
variant 'signal' | 'ghost' | 'field' | 'danger' 'signal' VISUAL VARIANT
size 'sm' | 'md' | 'lg' 'md' BUTTON SIZE
fieldEffect 'shimmer' | 'pulse' | 'glitch' | 'none' 'none' FIELD LAYER EFFECT
fieldIntensity number (0-1) 0.5 EFFECT STRENGTH
disabled boolean false DISABLE INTERACTION
onClick () => void — CLICK HANDLER REQ

USAGE

SIGNAL VARIANT (DEFAULT)

<Button variant="signal"> GET STARTED </Button>

FIELD VARIANT (GENERATIVE)

<Button variant="field" fieldEffect="shimmer" fieldIntensity={0.8} > LAUNCH SEQUENCE </Button>

ACCESSIBILITY

WCAG 2.1 AA COMPLIANT • FOCUS RING: 3PX SOLID, OFFSET 2PX • CONTRAST: 7:1 MIN (SIGNAL), 4.5:1 (FIELD) • FIELD EFFECTS: RESPECTS PREFERS-REDUCED-MOTION • ARIA: ROLE="BUTTON" AUTOMATIC
