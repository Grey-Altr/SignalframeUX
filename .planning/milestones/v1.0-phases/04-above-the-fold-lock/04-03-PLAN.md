Philosophy: reduced-motion is an intentional alternative design, not "animations off." Emphasis shifts to typography, spacing, and structure. Every effect resolves to its end-state.

| Error ScrambleText | Scramble on error code | Text visible immediately | `prefers-reduced-motion guard in useEffect` | Verified |

### Architecture

Two-layer suppression guarantees complete coverage:

1. **CSS layer (instant):** `@media (prefers-reduced-motion: reduce)` rules fire at paint time — before JavaScript loads. All `[data-anim]` elements reset to `opacity: 1; transform: none`. All decorative animations get `animation-duration: 0.01ms`.

2. **JS layer (runtime):** `gsap.globalTimeline.timeScale(0)` pauses all GSAP animations. `initHeroAnimations` is never called (early return in PageAnimations). Error page ScrambleText has explicit `matchMedia` guard.

3. **Runtime MQ change:** `initReducedMotion()` listens for runtime `prefers-reduced-motion` changes — if user toggles while page is open, GSAP timeline pauses/resumes accordingly.
