**2. SignalMesh component (components/animation/signal-mesh.tsx):**

Create a `'use client'` component that:

- Uses `useRef<HTMLDivElement>(null)` for the container element.
- Stores shader uniforms in `useRef` outside `buildScene` so ScrollTrigger can access them.
- Calls `useSignalScene(containerRef, buildScene)` with a factory that:
  - Creates `THREE.Scene` + `THREE.PerspectiveCamera(60, containerAspect, 0.1, 100)`, camera at z=2.5.
  - Computes aspect from `containerRef.current.clientWidth / containerRef.current.clientHeight` (fallback 1).
  - Creates `THREE.IcosahedronGeometry(1, 2)` (detail=2, 540 vertices).
  - Creates `THREE.ShaderMaterial` with uniforms `{ uTime: 0, uDisplacement: 0, uColor: resolveColorAsThreeColor("--color-primary"), uOpacity: 0.15 }`. The low uOpacity gives the "subtle solid fill at low opacity" per user decision.
  - Vertex shader: displaces vertices along normal using `sin(position.x * 4.0 + uTime * 0.8) * 0.5 + sin(position.y * 3.5 + uTime * 0.6) * 0.5` multiplied by `uDisplacement`. This creates organic breathing.
  - Fragment shader: `gl_FragColor = vec4(uColor, uOpacity)`.
  - Material: `transparent: true, side: THREE.DoubleSide, depthWrite: false`.
  - Creates `THREE.EdgesGeometry(geo)` + `THREE.LineSegments` with `THREE.LineBasicMaterial({ color: uniforms.uColor.value, transparent: true, opacity: 0.6 })`. This gives clean icosahedron edges (not triangulation diagonals). Per user decision: wireframe rendering with subtle solid fill.
  - Adds mesh and wireframe to scene. Returns `{ scene, camera }`.
  - Stores uniforms in `uniformsRef.current`.

- After scene registration, uses `useGSAP` with `{ scope: containerRef }` to:
  - Guard: `if (window.matchMedia("(prefers-reduced-motion: reduce)").matches) return;` ŌĆö no ScrollTrigger for reduced-motion.
  - Create `ScrollTrigger.create({ trigger: containerRef.current, start: "top bottom", end: "bottom top", onUpdate: (self) => { uniformsRef.current.uDisplacement.value = self.progress * 0.4; } })`.
  - Create a GSAP ticker callback that increments `uniformsRef.current.uTime.value += 0.016` (smooth rotation). Store the ticker function in a ref and return cleanup via `gsap.ticker.remove(tickerFn)`. Follow the ticker-accumulation-guard pattern from Phase 7: remove old ticker before adding new one.
  - Also slowly rotate the mesh: `meshRef.current.rotation.y += 0.003 * (1 + self.progress * 2)` inside the ticker or ScrollTrigger. Use a meshRef to store the mesh object.

- Camera aspect ratio handling: Use `ResizeObserver` on the container to update `camera.aspect` and call `camera.updateProjectionMatrix()`. Store camera in a ref.

- WebGL fallback: Wrap the canvas container div. If the component detects no WebGL support (`typeof window !== 'undefined' && !document.createElement('canvas').getContext('webgl2') && !document.createElement('canvas').getContext('webgl')`), render an SVG fallback instead. The SVG should be a static icosahedron wireframe silhouette using `<polygon>` elements in the primary color. Keep the SVG simple ŌĆö 12 vertices of an icosahedron projected to 2D with edge lines.

- Disposal logging: In the `buildScene` factory, after scene construction, log `console.debug("[SignalMesh] registered ŌĆö geometries:", getState().renderer?.info.memory.geometries)`. The `useSignalScene` cleanup will call `disposeScene` automatically.

- The component renders: `<div ref={containerRef} className="absolute inset-0 z-0" aria-hidden="true" />`. The SignalCanvas singleton reads the div's bounding rect for scissor/viewport.

- Export as named export: `export function SignalMesh()`.
