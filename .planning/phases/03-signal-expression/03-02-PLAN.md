Create `components/animation/canvas-cursor.tsx` as a `'use client'` component.

**Component structure:**
```typescript
"use client";
import { useEffect, useRef } from "react";
```

**Particle type:**
```typescript
type Particle = { x: number; y: number; alpha: number };
```

**CanvasCursor component:**
- Accepts no props. Returns a `<canvas>` element with:
  - `position: fixed; inset: 0; pointer-events: none; z-index: var(--z-cursor, 9999)`
  - `width` and `height` set to `window.innerWidth` and `window.innerHeight` via resize handler
- Uses `useRef` for: canvasRef, particles array, mousePos, isActive (section scoping), rafId

**useEffect logic (single effect, cleanup on unmount):**

1. **Touch guard:** `if (window.matchMedia("(pointer: coarse)").matches) return;` — exit early, no listeners
2. **Canvas setup:** Get 2D context, set canvas size to viewport, handle resize
3. **Mouse tracking:** `mousemove` listener updates mousePos ref and pushes new Particle `{ x, y, alpha: 1.0 }` to particles array
4. **Section scoping via IntersectionObserver:**
   - Query `document.querySelectorAll("[data-cursor]")`
   - Create IntersectionObserver with `threshold: 0` — on intersect entry, set `isActive.current = true`, on exit check if ANY observed section is still intersecting, if none set `isActive.current = false`
   - Track active section count with a ref counter (increment on enter, decrement on exit)
5. **Render loop (requestAnimationFrame):**
   - Clear canvas each frame
   - If `!isActive.current`, skip drawing (canvas stays clear)
   - **Draw crosshair** at mousePos: 4 lines extending 24px from center (48px total extent), 1px stroke, color from `getComputedStyle(document.documentElement).getPropertyValue("--color-primary").trim()` — read once on mount, not every frame
   - **Draw particles:** for each particle, draw a 2px circle with `rgba` using particle.alpha. Reduce alpha by 0.02 each frame. Remove particles where `alpha <= 0`. Cap particles array at 60 max (shift oldest).
   - Request next frame
6. **Tab visibility:** `visibilitychange` listener — on hidden: `cancelAnimationFrame(rafId.current)`, on visible: restart render loop
7. **Cleanup:** Remove all listeners, disconnect IntersectionObserver, cancel rAF

**Color reading:** Read `--color-primary` once on mount. Parse the OKLCH value into an RGB approximation for canvas. Use a helper: compute the RGB from the computed style (canvas 2D context does not support OKLCH directly). Simplest approach: create a temporary 1x1 canvas, set `fillStyle` to the computed property value, draw a pixel, read it back with `getImageData` to get RGB values. Cache this RGB for all subsequent draws.

**Canvas resize:** Listen to `resize` event, update canvas width/height. Use `devicePixelRatio` for sharp rendering: set canvas `width = innerWidth * dpr`, `height = innerHeight * dpr`, then `ctx.scale(dpr, dpr)`, and set CSS width/height to `100vw`/`100vh`.

**Export:** `export { CanvasCursor };`
