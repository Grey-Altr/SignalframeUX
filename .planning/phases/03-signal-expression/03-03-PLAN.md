```typescript
async function initPageHeadingScramble(headings: NodeListOf<Element>) {
  const { ScrambleTextPlugin } = await import("@/lib/gsap-plugins");
  if (!ScrambleTextPlugin) return;

  headings.forEach((el, i) => {
    const htmlEl = el as HTMLElement;
    const originalText = htmlEl.textContent || "";

    ScrollTrigger.create({
      trigger: htmlEl,
      start: "top bottom",  // fires even if already in viewport
      once: true,
      onEnter: () => {
        gsap.to(htmlEl, {
          duration: 0.8,
          delay: 0.1 + i * 0.05,
          scrambleText: {
            text: originalText,
            chars: "01!<>-_\\/[]{}—=+*^?#",
            speed: 0.4,
          },
        });
      },
    });
  });
}
```

Key changes:
- Remove `setTimeout(() => { ... }, 0)` wrapper entirely
- Wrap each heading's GSAP tween in a `ScrollTrigger.create({ once: true, onEnter })` block
- Use `start: "top bottom"` so above-the-fold headings trigger immediately
- Keep `delay: 0.1 + i * 0.05` for sequential stagger within a page's headings
- Keep scrambleText config (chars, speed, duration) unchanged

Also add the stagger batch to `initCoreAnimations` — see Task 2.
