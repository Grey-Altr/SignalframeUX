Create `components/animation/glsl-hero.tsx` as a 'use client' component following the exact SignalMesh pattern:

**Scene setup (buildScene factory):**
- OrthographicCamera(-1, 1, 1, -1, 0, 1) — fills NDC space
- PlaneGeometry(2, 2) — full-screen quad
- ShaderMaterial with uniforms stored in useRef (accessible from ScrollTrigger + ticker closures):
  - uTime: { value: 0 }
  - uScroll: { value: 0 }
  - uColor: { value: resolveColorAsThreeColor("--color-primary", { ttl: 2000 }) }
  - uGridDensity: { value: 12.0 }
  - uDitherOpacity: { value: 0.25 } (25% blend — middle of 20-30% range)
- transparent: true, depthWrite: false

**Vertex shader (pass-through):**
```glsl
void main() {
  gl_Position = vec4(position, 1.0);
}
```

**Fragment shader — single-pass with integrated dither:**
1. Compute UV from gl_FragCoord.xy / uResolution (add uResolution uniform, set from container size)
2. FBM noise (4 octaves): hash function `fract(sin(dot(p, vec2(127.1, 311.7))) * 43758.5453)`, accumulate with 0.5 amplitude decay per octave. Offset by uTime * 0.1 for slow drift. Scale modulated by uScroll (1.0 + uScroll * 2.0).
3. Grid lines: `step(fract(uv.x * uGridDensity), 0.02) + step(fract(uv.y * uGridDensity), 0.02)` — thin lines at regular intervals. Clamp to 1.0.
4. Combine: `float signal = noise * 0.6 + grid * 0.3`
5. Bayer 4x4 ordered dither (SCN-03):
```glsl
float bayer4x4[16] = float[16](
   0.0/16.0,  8.0/16.0,  2.0/16.0, 10.0/16.0,
  12.0/16.0,  4.0/16.0, 14.0/16.0,  6.0/16.0,
   3.0/16.0, 11.0/16.0,  1.0/16.0,  9.0/16.0,
  15.0/16.0,  7.0/16.0, 13.0/16.0,  5.0/16.0
);
int ix = int(mod(gl_FragCoord.x, 4.0));
int iy = int(mod(gl_FragCoord.y, 4.0));
float threshold = bayer4x4[iy * 4 + ix];
float dithered = step(threshold, signal);
```
6. Final color: `gl_FragColor = vec4(uColor * dithered, uDitherOpacity)` — monochrome using --color-primary at 25% opacity

**Scroll wiring (same pattern as SignalMesh):**
- ScrollTrigger.create on container element: `onUpdate: (self) => { uScroll.value = self.progress; uGridDensity.value = 12.0 + self.progress * 8.0; }`
- Start: "top bottom", end: "bottom top"

**Ticker (same pattern as SignalMesh with accumulation guard):**
- `gsap.ticker.remove(tickerFn)` then `gsap.ticker.add(tickerFn)` — HMR guard
- tickerFn increments uTime.value by 0.016

**Reduced-motion guard:**
- Check `matchMedia('(prefers-reduced-motion: reduce)')` at component level
- If reduced-motion: render a static div with background-color from --color-primary at 10% opacity, no WebGL
- Same pattern as SignalMesh SVG fallback

**Container element:**
- `<div ref={containerRef} className="absolute inset-0 z-0" aria-hidden="true" />` — same as SignalMesh
