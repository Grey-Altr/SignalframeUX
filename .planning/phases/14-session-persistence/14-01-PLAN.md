**hooks/use-scroll-restoration.ts** (~35 lines):
- Add 'use client' directive at top
- Export `useScrollRestoration()` (no params)
- Use `usePathname()` from `next/navigation` to build storage key: `sfux.scroll.${pathname}`
- First `useEffect` (save): add `beforeunload` event listener that saves `window.scrollY` to sessionStorage. Cleanup function both saves scroll position (SPA navigation away) and removes the event listener. Dependency: `[key]`

- Wrap all sessionStorage access in try/catch — silent ignore on failure
- JSDoc: note Lenis compatibility (rAF timing), note that sessionStorage clears on hard reload
