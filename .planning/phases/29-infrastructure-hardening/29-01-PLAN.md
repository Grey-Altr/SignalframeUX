    Five targeted edits, each 1-5 lines:

    **1. app/globals.css — overscroll-behavior: none**

    ```typescript

      duration: 1.2,

    });
    ```

    **3. lib/gsap-plugins.ts — Observer registration + export**
    Add import at line 7 (after the CustomEase import, before useGSAP):

    ```typescript
    import { Observer } from "gsap/Observer";
    ```

    Add `Observer` to the `gsap.registerPlugin(...)` call (after CustomEase, before useGSAP):

    ```typescript
    gsap.registerPlugin(
      ScrollTrigger,
      SplitText,
      ScrambleTextPlugin,
      Flip,
      CustomEase,
      Observer,
      useGSAP
    );
    ```

    Add `Observer` to the bottom export line:

    ```typescript
    export { gsap, ScrollTrigger, SplitText, ScrambleTextPlugin, Flip, CustomEase, Observer, useGSAP };
    ```

    **4. lib/gsap-core.ts — Observer re-export**
    Add import at line 9 (after ScrollTrigger, before useGSAP):

    ```typescript

    ```

    Add `Observer` to `gsap.registerPlugin(...)`:

    ```typescript
    gsap.registerPlugin(ScrollTrigger, Observer, useGSAP);
    ```

    Add `Observer` to the export line:

    ```typescript
    export { gsap, ScrollTrigger, Observer, useGSAP };
    ```

    **5. components/layout/page-animations.tsx — fonts-ready hook**
    Inside the `init()` function, AFTER the `ctx = gsap.context(() => { initCoreAnimations(clickCleanups); });` block (after line 44) and BEFORE the hero animations section (line 47), add:

    ```typescript
    // Font-ready ScrollTrigger recalculation — prevents CLS from font reflow
    // after Anton/Electrolize swap in. Must fire after GSAP context is established.
    document.fonts.ready.then(() => {
      if (!cancelledRef.current) ScrollTrigger.refresh();
    });
    ```

    The `cancelledRef.current` guard prevents refresh after component unmount (StrictMode safety). Do NOT add this inside the reduced-motion early-return branch (line 25-31) — no ScrollTrigger exists there, refresh would be a no-op.
