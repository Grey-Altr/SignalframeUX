2. **Pack the tarball**: Use `execFileSync('npm', ['pack', '--pack-destination', tmpDir])` to create the tarball in the temp directory. **CRITICAL: Capture the actual tarball filename dynamically from stdout** — `npm pack` prints the filename to stdout (e.g. `signalframeux-0.1.0.tgz`). Capture it via:
```typescript
const tarballName = execFileSync('npm', ['pack', '--pack-destination', tmpDir], { encoding: 'utf-8' }).trim();
```

6. **Run build**: `execFileSync('npm', ['run', 'build'], { cwd: tmpDir, stdio: 'pipe' })`. If this exits 0, the test passes.

7. **Cleanup**: In a `finally` block, `rmSync(tmpDir, { recursive: true, force: true })`.

8. **Output**: Print `Consumer test PASS: Next.js 16 build succeeded with SFUX imports from all 3 entry points` on success. On failure, print the stderr/stdout from the failed command and exit with code 1.
