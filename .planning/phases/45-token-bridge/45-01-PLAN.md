**Part B — Create scripts/wrap-tokens-layer.ts:**

```typescript
// Read lib/tokens.css, output dist/signalframeux.css
// @theme inline { } blocks stay OUTSIDE @layer
// :root { } and .dark { } blocks go INSIDE @layer signalframeux { }
```

Implementation approach:
1. Read lib/tokens.css as string
2. Parse into blocks: identify @theme, @custom-variant, :root, .dark by tracking brace depth
3. For @theme blocks and @custom-variant: emit as-is (outside layer)
4. For :root and .dark blocks: collect them, then emit wrapped in `@layer signalframeux { ... }`
5. Preserve all comments
6. Write to dist/signalframeux.css

Use a simple line-by-line state machine (not regex). Track `inTheme`, `inBlock`, `blockType`, `braceDepth`. When encountering `:root {` or `.dark {` at depth 0, start collecting. When brace depth returns to 0, stop collecting. Emit collected blocks inside the @layer wrapper.

Handle the header comment block — preserve it at the top of the output file.
