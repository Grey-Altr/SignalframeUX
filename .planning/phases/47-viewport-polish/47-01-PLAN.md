```css
--sfx-text-xs: clamp(9px, calc(0.694*var(--sf-vw)), 14px);   /* 10px at 1440px — tag pills, stat labels */
```

Replace with (raises min floor only — preserves the 0.625 / 0.694 multipliers and 12px / 14px caps so behavior above ~1600px is unchanged):
```css
--sfx-text-2xs: clamp(10px, calc(0.625*var(--sf-vw)), 12px); /* 10px floor — readable on 1280px MBP — micro labels, copy buttons */
--sfx-text-xs: clamp(11px, calc(0.694*var(--sf-vw)), 14px);  /* 11px floor — readable on 1280px MBP — tag pills, stat labels */
```

Math: with multiplier 0.625, calc reaches 10px at vw=16 (viewport=1600px). With multiplier 0.694, calc reaches 11px at vw=15.85 (viewport=1585px). Below those viewports, clamp pins at the new min floor. Pure floor lift — no curve change.
