3. **GLSL shaders** — inline as string constants:
   - Vertex: `void main() { gl_Position = vec4(position, 1.0); }`
   - Fragment shader (the datamosh effect):
     ```glsl
     precision highp float;
     uniform float uTime;
     uniform float uIntensity;
     uniform float uBlockRes;
     uniform vec2 uResolution;

     float hash(vec2 p) {
       return fract(sin(dot(p, vec2(127.1, 311.7))) * 43758.5453);
     }

     float noise(vec2 p) {
       vec2 i = floor(p);
       vec2 f = fract(p);
       vec2 u = f * f * (3.0 - 2.0 * f);
       return mix(
         mix(hash(i), hash(i + vec2(1.0, 0.0)), u.x),
         mix(hash(i + vec2(0.0, 1.0)), hash(i + vec2(1.0, 1.0)), u.x),
         u.y
       );
     }

     void main() {
       vec2 uv = gl_FragCoord.xy / uResolution;
       float n = noise(uv * 8.0 + vec2(uTime * 0.3, uTime * 0.17));
       vec2 offset = vec2(n, noise(uv * 8.0 + vec2(uTime * 0.17, uTime * 0.3))) * uIntensity;
       vec2 blockUV = floor(uv * uBlockRes) / uBlockRes;
       vec2 sampleUV = mix(uv, blockUV, step(0.003, uIntensity) * 0.15);
       vec2 uvR = sampleUV + offset;
       vec2 uvG = sampleUV + offset * 0.66;
       vec2 uvB = sampleUV - offset * 0.33;
       float r = noise(uvR * 12.0 + uTime * 0.05) * uIntensity * 40.0;
       float g = noise(uvG * 12.0 + uTime * 0.05) * uIntensity * 40.0;
       float b = noise(uvB * 12.0 + uTime * 0.05) * uIntensity * 40.0;
       gl_FragColor = vec4(r, g, b, max(r, max(g, b)));
     }
     ```

4. **DatamoshFallback** — returns `<div data-signal-scene="datamosh-static" className="fixed inset-0 pointer-events-none" aria-hidden="true" />`

   - `uniformsRef` with typed uniforms: `uTime`, `uIntensity` (default 0.003), `uBlockRes` (default 256.0), `uResolution`

   - **ResizeObserver for uResolution** (same pattern as GLSLSignal):
     ```typescript
     useEffect(() => {
       const container = containerRef.current;
       if (!container || !uniformsRef.current) return;
       const ro = new ResizeObserver((entries) => {
         for (const entry of entries) {
           const { width, height } = entry.contentRect;
           if (width > 0 && height > 0 && uniformsRef.current) {
             uniformsRef.current.uResolution.value.set(width, height);
           }
         }
       });
       ro.observe(container);
       return () => ro.disconnect();
     }, []);
     ```
   - `useGSAP` block:
     - Guard: if reduced-motion, return early
     - Call `ensureSignalObserver()`
     - Create breathing object: `{ value: 0.003 }`
     - Create breathing tween: `gsap.to(breathingObj, { value: 0.001, duration: gsap.utils.random(8, 12), ease: "sine.inOut", yoyo: true, repeat: -1 })`
     - **Ticker function — CORRECTED uIntensity formula (per xtop spec: max 0.006):**
       ```typescript
       const tickerFn = () => {
         if (!uniformsRef.current) return;
         uniformsRef.current.uTime.value += 0.016;
         // Formula: breathingBase (0.001-0.004) + signalIntensity * 0.002
         // At intensity=0.0: range 0.001-0.004 (breathing only)
         // At intensity=0.5: range 0.002-0.005 (default)
         // At intensity=1.0: range 0.003-0.006 (max ceiling per spec)
         // This guarantees uIntensity never exceeds 0.006
         uniformsRef.current.uIntensity.value =
           breathingObj.value + _signalIntensity * 0.002;
       };
       ```
     - HMR guard: `gsap.ticker.remove(tickerFn)` before `gsap.ticker.add(tickerFn)`
     - Cleanup: remove ticker, kill breathing tween, disconnect MutationObserver + null it (TD-01)
   - Render: `<div ref={containerRef} data-signal-scene="datamosh-overlay" className="fixed inset-0 pointer-events-none" style={{ zIndex: -1 }} aria-hidden="true" />`

   - `hasWebGL` state via checkWebGL
   - `isReducedMotion` state with matchMedia listener (copy from glsl-signal.tsx pattern)
   - If no WebGL or reduced motion: return `<DatamoshFallback />`

**CRITICAL — uIntensity formula rationale (per user decision, CONTEXT.md locked spec):**
The xtop spec locks `uIntensity = signalIntensity * 0.006` (max 0.006 at full intensity). The breathing oscillation (0.001-0.004) is the *base*, and signal intensity adds up to 0.002 on top. The formula `breathingObj.value + _signalIntensity * 0.002` ensures:
- breathing range at idle (intensity=0): 0.001 to 0.004
- breathing range at default (intensity=0.5): 0.002 to 0.005
- breathing range at full (intensity=1.0): 0.003 to 0.006
- **Max uIntensity = 0.004 + 1.0 * 0.002 = 0.006** -- exactly the xtop ceiling

DO NOT use the formula `breathingObj.value * (1 + _signalIntensity * 2)` — that reaches 0.012 at full intensity, violating the spec.
