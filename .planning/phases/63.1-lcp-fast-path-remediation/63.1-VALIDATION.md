## Sampling Rate

- **After every task commit:** Run `pnpm test:smoke` (smoke + tsc --noEmit + lint).
- **After every plan wave:** Run `pnpm lhci:run` (LHCI median-of-5 against preview deployment) + relevant Playwright spec subset.
- **Before `/pde:verify-work`:** Full suite must be green AND `pnpm wpt:vrf-01-post-63.1` (re-run vrf-01-* WPT scripts on all 3 profiles, all LCP <1000 ms) must pass.
- **Max feedback latency:** 45s for smoke; 4m for LHCI; 8–15m for WPT (3 profiles × ~3–5m each).
