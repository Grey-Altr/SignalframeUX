**T-RUM-03 (LOW) — Branch-scope coupling**
Aggregator silently scopes to current git branch when `--no-branch` flag is omitted (vercel CLI default), causing zero-result queries on feature branches with no prod deploys (the documented v1.8 VRF-05 deferral cause).
**Mitigation:** `--no-branch` flag asserted in unit test (`70-01-no-branch`); `--deployment <prod-url>` flag mandatory and asserted via argv check.

**T-PARSE-01 (LOW) — UA parser shape ambiguity**
Vercel Log Drains schema documents `proxy.userAgent` as array; current `vercel logs --json` CLI output flattens to string. Schema drift between sources silently fails UA cohort matching (mobile/desktop split shows `mobile: 0, desktop: 105`).
**Mitigation:** Defensive `uaString()` polymorphic helper handles both shapes; unit test `70-01-ua-parser` covers array, string, and undefined inputs.
</threat_model>

          p75_lcp_ms: percentile(iosLcp, 75),
          threshold_median_ms: 2000,
          verdict: iosVerdict,
        },
        thresholds: { p75_lcp_ms_max: 1000, sample_count_min: 100 },
        sample_source: opts.sampleSource,
