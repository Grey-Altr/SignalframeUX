### Partial Requirements Detail

**INT-03 — SignalMotion (zero consumers):**
`signal-motion.tsx` is a fully implemented 102-line component with scrub support, GSAP ScrollTrigger, and reduced-motion guard. However, zero pages or blocks import or use `<SignalMotion>`. The requirement states it "provides scroll-driven motion graphics for showcase sections" — without placement, that intent is unfulfilled. The component is a shipped capability with no activation.

**INT-04 — SIGNAL Overlay (incomplete feedback loop):**
`signal-overlay.tsx` works as UI: Shift+S opens it, 3 sliders move, CSS vars are written to `:root`. But `--signal-intensity`, `--signal-speed`, and `--signal-accent` are not read by `glsl-hero.tsx` or `signal-mesh.tsx`. Adjusting "Signal Intensity" produces no visible change to any generative surface. The overlay is shipped chrome without downstream effect.
